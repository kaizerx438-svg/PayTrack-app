# PayTrack-app
API REST Laravel pour la gestion des factures et des clients avec une architecture orientée métier.

PayTrack est une API REST axée sur le backend et développée avec Laravel.
L'objectif de ce projet est de mettre en pratique une architecture backend propre, la conception de bases de données 
et la mise en œuvre d'une logique métier autour d'un système de facturation.

## Fonctionnalités
- Authentification des utilisateurs via l'API
- Gestion des clients (particuliers et entreprises)
- Création de factures et gestion du cycle de vie
- Système de numérotation des factures
- Points de terminaison pour les revenus et les statistiques
- Application des règles métier (statut, immuabilité, etc.)

## pile technologique
- Laravel
- MySQL
- Eloquent ORM
- Laravel Sanctum
- API REST

## Objectif
Ce projet est destiné à la formation au backend et à la mise en pratique de l'architecture.
Il ne comprend pas de mise en œuvre complète du frontend.
